## Staff Augmentation

App Agent is designed to be self-service, but additional support is available for customers who need it.

## Available Services

### Setup Assistance

Help with initial deployment and configuration:
- Cloud infrastructure provisioning
- CI/CD pipeline configuration
- Domain and SSL setup
- Initial customization

### Architecture Guidance

Design reviews and best practices:
- Feature design consultation
- Code review and feedback
- Performance optimization
- Security review

### Feature Development

Build features using the AI-native workflow:
- Pair programming with AI
- Complex feature implementation
- Integration work
- Migration assistance

### Training

Onboard your team to the platform:
- Developer onboarding sessions
- AI workflow training
- Best practices workshops
- Custom documentation

## How It Works

1. **Fork the repo** - You own everything from day one
2. **Request support** - Contact for specific needs
3. **Collaborate** - Work together using the AI-native tools
4. **Own the result** - All work goes into your repo

## Pricing

Contact for current pricing and availability.

Services are billed separately from the platform itself. The platform is MIT licensed and free to use - services are optional.

## Self-Service First

Most customers don't need services. The platform is designed to be:

- **Self-documenting** - MCP gives AI full context
- **Pattern-based** - Copy demos, modify for your needs
- **Convention-enforced** - Hard to make mistakes
- **Continuously updated** - Pull upstream improvements

Try the self-service approach first. Services are available when you need acceleration or expertise.
