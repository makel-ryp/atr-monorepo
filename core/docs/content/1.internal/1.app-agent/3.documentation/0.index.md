## Overview

This section covers how to write documentation. The docs are built with [Nuxt Content](https://content.nuxt.com) and the [Nuxt UI Docs Template](https://docs-template.nuxt.dev/).

## Documentation Structure

Documentation lives in `core/docs/content/`:

```
core/docs/content/
├── index.md                   # Homepage
├── 1.internal/
│   ├── 0.index.md            # Internal docs landing
│   └── 1.app-agent/          # App Agent reference
│       ├── 1.getting-started/
│       ├── 2.demos/
│       ├── 3.essentials/     # This section
│       ├── 4.ai/
│       └── 5.services.md
└── 2.your-docs/              # Add your docs here
```

## Adding Pages

Create `.md` files in `core/docs/content/`:

```md [core/docs/content/2.my-docs/1.index.md]
---
title: My Docs
description: What this section covers
---

Your content here...
```

The number prefix controls sort order. Navigation is generated automatically.

## Organizing Content

Use numbered prefixes to control order:

```
content/
├── index.md              # Homepage (no prefix)
├── 1.internal/           # First section
├── 2.my-docs/            # Second section
└── 3.api/                # Third section
```
