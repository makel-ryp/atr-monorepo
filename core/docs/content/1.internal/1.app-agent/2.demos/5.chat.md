# Chat Demo

The Chat demo showcases patterns for building AI-powered conversational interfaces with persistent history and authentication.

## Features

- **AI Chatbot**: Conversational interface powered by Vercel AI SDK
- **Multiple Providers**: Support for Claude, OpenAI, and other LLMs via AI Gateway
- **Authentication**: GitHub OAuth login
- **Persistent History**: Chat sessions saved to database
- **File Attachments**: Upload files to conversations via Vercel Blob
- **Markdown Rendering**: Rich text responses with code highlighting

## Running the Demo

```bash
# Run all demos
bun run dev:demos

# Visit http://localhost:3013
```

## Environment Variables

This demo requires configuration. Copy `.env.example` to `.env`:

```bash
cp demos/chat/.env.example demos/chat/.env
```

Required variables:

| Variable | Description |
|----------|-------------|
| `NUXT_SESSION_PASSWORD` | Session encryption key (min 32 chars) |
| `NUXT_OAUTH_GITHUB_CLIENT_ID` | GitHub OAuth app client ID |
| `NUXT_OAUTH_GITHUB_CLIENT_SECRET` | GitHub OAuth app client secret |
| `AI_GATEWAY_API_KEY` | Vercel AI Gateway API key |
| `DATABASE_URL` | PostgreSQL connection string (optional - uses SQLite by default) |

## Key Patterns

### AI SDK Integration

Uses Vercel AI SDK for streaming responses:

```typescript
import { useChat } from '@ai-sdk/vue'

const { messages, input, handleSubmit, isLoading } = useChat({
  api: '/api/chat'
})
```

### Database Schema

Chat history persisted with Drizzle ORM:

```typescript
// server/database/schema.ts
export const chats = sqliteTable('chats', {
  id: text('id').primaryKey(),
  userId: text('user_id').notNull(),
  title: text('title'),
  createdAt: integer('created_at', { mode: 'timestamp' })
})
```

### Authentication

GitHub OAuth via nuxt-auth-utils:

```typescript
// Protect routes
definePageMeta({
  middleware: 'auth'
})

// Access user
const { user } = useUserSession()
```

## Customizing

To use this demo as a starting point:

1. Copy to `/apps/your-app-name`
2. Update `package.json` with your app name
3. Configure environment variables
4. Run database migrations: `bun run db:migrate`
5. Customize the AI provider and prompts

## Tech Stack

- **Vercel AI SDK** - Streaming AI responses
- **NuxtHub** - Database and blob storage
- **Drizzle ORM** - Type-safe database access
- **nuxt-auth-utils** - OAuth authentication
- **Nuxt MDC** - Markdown rendering
