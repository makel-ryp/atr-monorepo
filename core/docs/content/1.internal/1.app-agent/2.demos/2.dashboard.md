# Dashboard Demo

The Dashboard demo showcases patterns for building internal tools, admin panels, and analytics dashboards.

## Features

- **Authentication**: Login/logout flows with session management
- **Data Tables**: Sortable, filterable tables using TanStack Table
- **Charts**: Analytics visualizations with Unovis
- **Settings**: User preferences and app configuration
- **API Routes**: Server-side data fetching and mutations

## Running the Demo

```bash
# Run just the dashboard
bun run dev:demos

# Visit http://localhost:3010
```

## Directory Structure

```
demos/dashboard/
├── app/
│   ├── components/    # Dashboard-specific components
│   ├── composables/   # Data fetching and state
│   ├── layouts/       # Admin layout with sidebar
│   └── pages/         # Dashboard pages
├── server/
│   └── api/           # API endpoints
└── nuxt.config.ts     # Extends core layer
```

## Key Patterns

### API Integration

The dashboard demonstrates how to create API routes in `server/api/` and consume them from the frontend.

### State Management

Uses Vue composables for local state and server state with `useFetch`.

### Layout System

Shows how to create authenticated layouts with sidebars and navigation.

## Customizing

To use this demo as a starting point:

1. Copy to `/apps/your-app-name`
2. Update `package.json` with your app name
3. Modify components and pages as needed
4. The app automatically extends `/core` for shared functionality
