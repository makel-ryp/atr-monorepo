# Demos

App Agent includes four demo applications that showcase different use cases and patterns. These demos are designed to be starting points for your own applications.

## Using Demos

### Explore First

Run all demos to explore the patterns:

```bash
bun run dev:demos
```

### Copy to Start Your App

When you run `bun run dev` with an empty `/apps` folder, you'll be prompted to copy a demo as your starting point.

Or manually:

```bash
cp -r demos/dashboard apps/my-app
```

Then update `apps/my-app/package.json` with your app name and port.
