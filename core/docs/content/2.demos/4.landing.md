# Landing Demo

The Landing demo showcases patterns for building marketing sites, documentation, and landing pages with SEO optimization.

## Features

- **Landing Pages**: Marketing-focused page layouts
- **Documentation**: Built with Nuxt Content and MDC
- **SEO**: Meta tags, sitemap, and structured data
- **OG Images**: Automatic social media preview images

## Running the Demo

```bash
# Run all demos
bun run dev:demos

# Visit http://localhost:3012
```

## Directory Structure

```
demos/landing/
├── app/
│   ├── components/    # Marketing components
│   ├── layouts/       # Landing and docs layouts
│   └── pages/         # Landing and docs pages
├── content/           # Documentation content
├── public/            # Static assets
└── nuxt.config.ts     # Extends core layer
```

## Key Patterns

### SEO Optimization

Uses `nuxt-og-image` for automatic OG image generation and proper meta tags.

```typescript
// In nuxt.config.ts
modules: ['nuxt-og-image']
```

### Content-Driven Pages

Documentation pages are written in Markdown with MDC components.

### Pre-rendering

Static pages are pre-rendered for optimal performance:

```typescript
nitro: {
  prerender: {
    routes: ['/'],
    crawlLinks: true
  }
}
```

## Customizing

To use this demo as a starting point:

1. Copy to `/apps/your-app-name`
2. Update `package.json` with your app name
3. Replace content in `/content/` with your docs
4. Customize landing page components
