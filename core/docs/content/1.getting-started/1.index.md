App Agent is a turnkey Nuxt 4 monorepo template designed for companies to fork and customize. It provides a solid foundation with shared components, multiple demo applications, and AI-native documentation.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/your-org/app-agent.git
cd app-agent

# Install dependencies
bun install

# Start development
bun run dev
```

If `/apps/` is empty, you'll be prompted to copy a demo as your starting point.

## Key Features

- **Nuxt 4** with the latest `app/` directory structure
- **Nuxt UI v4** - 125+ free components (MIT licensed)
- **Turborepo** for fast, cached monorepo builds
- **Bun** as package manager and runtime
- **TypeScript** throughout with strict mode
- **MCP Server** for AI assistant integration
- **Demo Apps** showing real-world patterns

## Project Structure

```
app-agent/
├── core/              # Shared Nuxt layer
│   ├── components/    # Shared UI components
│   ├── composables/   # Shared composables
│   ├── docs/          # This documentation
│   └── cli/           # Dev tooling
├── demos/             # Reference implementations
│   ├── dashboard/     # Internal tools pattern
│   ├── saas/          # Customer app pattern
│   └── landing/       # Marketing site pattern
├── apps/              # Your applications (empty initially)
└── packages/          # Optional shared packages
```
