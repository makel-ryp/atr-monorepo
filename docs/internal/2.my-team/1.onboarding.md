# Onboarding Guide

Welcome to the team! This guide will help you get started.

## First Day

1. Set up your development environment
2. Clone the repository
3. Run `bun install`
4. Run `bun run dev`

## First Week

- Review the [App Agent documentation](/internal/app-agent/getting-started)
- Explore the demo applications
- Set up your first customer app

- Slack: #engineering
- Wiki: Internal confluence
