# My Team Documentation

This is an example of customer internal documentation that appears alongside the upstream app-agent docs in the `/internal/` section.

## Why This Location?

Content in `/docs/internal/` merges into the existing internal navigation:
- `/internal/app-agent/` - Upstream documentation (don't edit)
- `/internal/my-team/` - Your team's docs (this folder)

## Getting Started

Add your team's documentation here. Use numbered prefixes to control navigation order:

```
/docs/internal/2.my-team/
├── 0.index.md        # This file (section landing)
├── 1.onboarding.md   # First nav item
└── 2.processes.md    # Second nav item
```
